# bitcommerce-api
api for building a bitcoin shop

## To serve the project:
1. Run npm install
2. ts-node ./src/app.ts (or nodemon ./src/app.ts if you have nodemon installed)
3. NOTE: Please use Node version 18.15.0 or generate invoice won't work!
